# InnoviQR - Advanced QR Code Generator

A modern, feature-rich QR code generator with advanced styling options and multiple output formats.

## üöÄ Features

### Content Types
- **Plain Text** - Any text content
- **Website URL** - Web links with validation
- **Email Address** - Mailto links with subject and body
- **Phone Number** - Tel links for direct calling
- **WiFi Network** - WiFi credentials with security options
- **WhatsApp Message** - Pre-filled WhatsApp messages
- **Contact Card** - vCard format for contact information

### Basic QR Generation
- Adjustable size (200px - 800px)
- Error correction levels (Low, Medium, Quartile, High)
- Clean black and white QR codes
- Fast generation using qrcode.js library

### Advanced Styling Options
- **Colors & Gradients**
  - Single color QR codes
  - Linear and radial gradients
  - Custom background colors
  - Transparent backgrounds

- **Dot Styles**
  - Square (default)
  - Dots
  - Rounded squares
  - Extra rounded
  - Classy styles

- **Corner Styles**
  - Square corners
  - Dot corners
  - Extra rounded corners

- **Logo Integration**
  - Upload custom logos (PNG, JPG, SVG)
  - Adjustable logo size (10% - 50%)
  - Logo margin controls
  - Automatic background removal for better scanning

### Download Options
- **PNG** - High-quality raster images (256px - 2048px)
- **SVG** - Vector graphics for infinite scalability
- **PDF** - Print-ready format with proper sizing
